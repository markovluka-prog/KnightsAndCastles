# Рыцари и Замки — HTML / JavaScript

![Knights and Castles](Knights_and_Castles_1920x1080.png)

Тактическая пошаговая стратегия. Браузерная версия на чистом HTML5/Canvas.

## Запуск

Открыть `index.html` в любом браузере — установка не нужна.

## Режимы

- **2 Игрока** — локальная игра на одном экране
- **Против ИИ** — игра против тактического ИИ-противника
